Signed APK package feed for OpenWrt {{VERSION}}.x and newer.

Install

On the router (OpenWrt {{VERSION}}.x+, apk-based):

ARCH=$(apk --print-arch)
wget -O /etc/apk/keys/luci-singbox.pem {{PAGES_URL}}/luci-singbox.pem
echo "{{PAGES_URL}}/{{VERSION}}/$ARCH/luci-singbox" > /etc/apk/repositories.d/luci-singbox.list
apk update && apk add luci-singbox-ui

OpenWrt 24.10.x and older use .ipk and are not served here — use the GitHub Releases artifacts instead.
